install ADCS Role

Entreprise CA -> Root CA -> Create New private key -> Next -> Write CA Name -> 5 Years -> Next -> Configure

Make a crl 

Install IIS Role

Create new website : http / crl.lyon2024.local / C:\Windows\System32\CertSrv

Request Filtering -> Edit Feature Settings -> Select Allow double escaping

Add new DNS : crl.lyon2024.local

Open Certification Authority 

Right Click on the name of CA -> Properties

Extensions : 

- CDP :  Remove file: & ldap : and http:
Add -> http://crl.lyon2024.local/CertEnroll/<CaName><CRLNameSuffix><DeltaCRLAllowed>.crl -> Select 3rd and 4th line

- AIA : Remove file: & ldap : and http:
Add -> http://crl.lyon2024.local/CertEnroll/<ServerDNSName>_<CaName><CertificateName>.crt -> Select 1st

Apply

Right click on Revoked Certificate -> Publish -> New 

For verify, start pkiview.msc -> ALL is OK

IF CRL is on CORE SERVER

install iis service on core server
regedit 
Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WebManagement\Server
EnableRemoteManagement -> 1
iisreset.exe
Start-Service WMSVC

You can modify the configuration of core server since gui server with server manager
