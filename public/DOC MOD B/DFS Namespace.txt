DFS namespace
- Two nodes will support this namespace: PARDI01 and LYREM02.
Install File and Storage Service -> DFS NameSpaces 
Server Manager -> DFS Management 
Namespaces -> New Namespace -> choice server -> name : shares -> next -> create
Click on the namespace -> Add Namespacer server -> browse -> select the target server

- Clients cache duration need to be set at 50 seconds.
Click on namespace -> properties -> referrals -> cache duration 50
- Users can view only folders that are accessible for him.
Click on namespace -> properties -> advanced -> enable access-based enumeration for this namespace
